Themes

CalPaper supports multiple theme styles to match user taste and workspace mood.

Available Theme Types

  • Minimal: Clean layout with low-contrast accents for distraction-free work.
  • Vibrant: High-energy colors and stronger contrast for visual impact.
  • Nature: Organic palette and soft gradients inspired by landscapes.
  • Night: Dark-first option with reduced glare for low-light usage.

Customization

Users can switch theme presets from the in-app settings, and any custom adjustments are saved locally on the device.

Best Practices

  • Use higher contrast for accessibility when text density is high.
  • Avoid over-saturated backgrounds behind core reading areas.
  • Keep a consistent accent color for primary interactions.
